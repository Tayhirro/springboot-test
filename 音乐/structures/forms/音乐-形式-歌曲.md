# 音乐-形式-歌曲

## 1. 一句话
- “歌曲”通常指可唱的短结构作品，常见模块是：主歌（Verse）/副歌（Chorus）/桥段（Bridge）/前副歌（Pre-chorus）。

## 2. 常见结构
- `Verse -> Chorus`（循环）
- `Verse -> Pre -> Chorus`
- `Verse -> Chorus -> Verse -> Chorus -> Bridge -> Chorus`
