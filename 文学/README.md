# 文学（Literature）知识库组织说明（可扩展 & 速查）

这部分按"面向对象/类型系统"的方式组织文学概念：
1) **结构族（structures）**：体裁/流派/形式（诗歌、小说、戏剧；浪漫主义、现实主义；十四行诗、律诗……）
2) **横切模块（modules）**：叙事技巧、修辞手法、主题母题、人物塑造……（跨作品/流派可复用）
3) **作品（works）**：经典文本分析（类比神经网络的 models/）
4) **作家（authors）**：作家生平与风格

---

## 1. 目录结构（入口 → 索引 → 概念图 → 结构 → 模块 → 作品/作家）
- `文学/README.md`：入口与组织方式（本页）
- `文学/文学索引.md`：层次化索引（中文｜英文｜一句话｜链接）
- `文学/概念图.md`：概念依赖图（"先学什么 → 再学什么"）

核心内容：
- `文学/structures/`：体裁/流派/形式的"类定义"
  - `structures/genres/`：体裁（诗歌、小说、散文、戏剧）
  - `structures/periods/`：时期/流派（古典主义、浪漫主义、现实主义、现代主义……）
  - `structures/forms/`：具体形式（十四行诗、律诗、章回体……）
- `文学/modules/`：横切概念与工具箱
  - 叙事技巧（视角、时间、结构）
  - 修辞手法（比喻、象征、反讽）
  - 主题母题（成长、爱情、死亡、身份）
  - 人物塑造（圆形人物、扁平人物、可靠叙述者）
- `文学/works/`：作品分析卡片（背景、结构、技巧、主题、影响）
- `文学/authors/`：作家卡片（生平、风格、代表作、影响）

资料：
- `文学/examples/`：最小分析例子（一个文本片段 + 分析思路）

## 2. 继承关系（面向对象视角）

### 2.1 体裁继承树（Genre Hierarchy）
```
文学作品（Literary Work）
├── 诗歌（Poetry）
│   ├── 抒情诗（Lyric Poetry）
│   ├── 叙事诗（Narrative Poetry）
│   └── 戏剧诗（Dramatic Poetry）
├── 散文（Prose）
│   ├── 小说（Fiction）
│   │   ├── 长篇小说（Novel）
│   │   ├── 中篇小说（Novella）
│   │   └── 短篇小说（Short Story）
│   └── 非虚构（Non-Fiction）
│       ├── 散文（Essay）
│       ├── 传记（Biography）
│       └── 游记（Travel Writing）
└── 戏剧（Drama）
    ├── 悲剧（Tragedy）
    ├── 喜剧（Comedy）
    └── 悲喜剧（Tragicomedy）
```

### 2.2 流派演化链（Period/Movement Evolution）
```
古典主义（Classicism）
    ↓
浪漫主义（Romanticism）
    ↓
现实主义（Realism）→ 自然主义（Naturalism）
    ↓
现代主义（Modernism）→ 后现代主义（Postmodernism）
```

## 3. 每个概念页的固定模板（像看类库文档）

### 结构（体裁/流派）模板
1) **一句话定义**：它是什么，核心特征
2) **接口：要素与规则**：必要成分、形式约束
3) **继承与对比**：父类/子类/相近概念的区别
4) **典型例子/反例**：代表作品 + 边界案例
5) **常见变体**：亚类型、跨界形式
6) **分析工具**：分析此类作品时重点关注什么

### 横切模块模板
1) **一句话定义**：是什么/干什么
2) **定义与分类**：细分类型、术语辨析
3) **直觉**：为什么这样设计/使用
4) **在哪些作品/体裁里出现**：典型用例
5) **速查**：关键词、常见误区

### 作品分析模板
1) **一句话**：作品是什么/为什么重要
2) **基本信息**：作者、年代、体裁
3) **内容概要**：情节/结构大纲
4) **核心技巧**：使用了哪些模块（叙事/修辞/主题）
5) **主题分析**：作品在说什么
6) **文学史定位**：影响与评价
7) **参考阅读**

## 4. 你接下来怎么用
- **速查**：从 `文学索引.md` 或 `概念图.md` 进，跳到具体页面
- **系统学**：按继承树读体裁/流派，每页只记"新增了什么特征"
- **分析作品**：优先看 `modules/`（叙事/修辞/主题），这些是分析工具箱
- **研究作家**：从 `authors/` 进，看风格特征与代表作
