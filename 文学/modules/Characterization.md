# 人物塑造（Characterization）

## 1. 一句话
- 作者创造和呈现文学人物的全部技巧，包括外在描写和内在揭示

## 2. 定义与分类
- **定义**：人物塑造是使虚构人物"活起来"的方法总和，让读者感知人物的外表、性格、动机、心理

- **按方法分类**：
  - **直接塑造**（Direct/Explicit Characterization）：
    - 叙述者直接告诉读者人物是什么样的
    - "他是个吝啬的人"
  - **间接塑造**（Indirect/Implicit Characterization）：
    - 通过展示让读者自己判断
    - STEAL 法则：
      - **S**peech（言语）：人物说什么、怎么说
      - **T**houghts（思想）：人物想什么
      - **E**ffect（影响）：人物对他人的影响
      - **A**ctions（行动）：人物做什么
      - **L**ooks（外表）：人物长什么样

- **按复杂度分类**（E.M. 福斯特）：
  - **扁平人物**（Flat Characters）：
    - 围绕单一特征/观念建构
    - 可以用一句话概括
    - 容易辨认、有类型化功能
  - **圆形人物**（Round Characters）：
    - 复杂、多面、有深度
    - 能够令人信服地发展变化
    - 不可预测但又合情合理

- **按功能分类**：
  - **主人公**（Protagonist）：中心人物
  - **对手**（Antagonist）：与主人公对立
  - **陪衬人物**（Foil）：通过对比突显主人公
  - **股票人物**（Stock Character）：类型化、可预测

## 3. 直觉（为什么这样设计/使用）
- **直接 vs 间接**：
  - 直接：快速建立印象，但可能不够生动
  - 间接：更有说服力，让读者参与，但需要更多篇幅
  - 现代小说倾向"展示而非告诉"（Show, don't tell）

- **扁平 vs 圆形**：
  - 扁平人物不是缺点：讽刺小说、次要角色需要
  - 圆形人物让读者认同、但塑造成本高
  - 同一作品可以混用

## 4. 在哪些作品/体裁里出现

### 圆形人物典范
- **《安娜·卡列尼娜》**的安娜：复杂的心理、矛盾的动机、真实的发展
- **《包法利夫人》**的爱玛：幻想与现实的冲突、逐渐堕落
- **《哈姆雷特》**：犹豫不决、复杂的内心世界

### 扁平人物/类型人物
- **狄更斯**的许多配角：夸张、漫画化、但难忘
- **侦探小说**的侦探：核心特征突出
- **民间故事/神话**的角色：原型化

### 间接塑造典范
- **海明威**："冰山理论"，大量留白
- **契诃夫**：通过细节和对话暗示

### 直接塑造
- **19世纪小说**：叙述者常直接评价人物
- **奥斯汀**：反讽的直接评价

## 5. 分析方法
- **如何分析人物塑造**：
  1. 人物是通过什么方式呈现的？（直接/间接）
  2. 人物是扁平还是圆形？是否发展变化？
  3. 使用了哪些 STEAL 技巧？
  4. 人物的核心特征/矛盾是什么？
  5. 人物在作品中的功能是什么？

- **分析维度**：
  - 外在：外貌、言行、社会身份
  - 内在：动机、心理、价值观
  - 关系：与其他人物的关系
  - 发展：从开始到结束的变化

  - 忽视次要人物的塑造功能
  - 混淆人物与真实人物（人物是作者构建的）
  - 只关注心理而忽视外在呈现
