# 叙事视角（Narrative Point of View）

## 1. 一句话
- 决定"谁在讲故事"以及"叙述者知道多少"的叙事策略

## 2. 定义与分类
- **定义**：叙事视角是叙述者与故事世界的关系，决定了读者能获取什么信息、从谁的角度体验故事

- **按人称分类**：
  - **第一人称**（First Person）：叙述者是故事中的人物，用"我"讲述
    - 主人公叙述者（Protagonist Narrator）：讲自己的故事
    - 旁观者叙述者（Witness Narrator）：讲别人的故事
  - **第二人称**（Second Person）：用"你"称呼主人公（少见）
  - **第三人称**（Third Person）：叙述者在故事之外
    - 全知视角（Omniscient）：知道所有人物的想法
    - 有限视角（Limited）：只知道某个人物的想法
    - 客观视角（Objective）：只描述外在行为

- **按知识量分类**（热奈特 Genette）：
  - **零聚焦**（Zero Focalization）：叙述者 > 人物（全知）
  - **内聚焦**（Internal Focalization）：叙述者 = 人物（有限）
  - **外聚焦**（External Focalization）：叙述者 < 人物（客观）

- **术语辨析**：
  - Narrator vs Author：叙述者 ≠ 作者（即使第一人称）
  - Voice vs Focalization：谁在说 vs 谁在看

## 3. 直觉（为什么这样设计/使用）
- **第一人称**：亲密感、可靠性问题、限制信息
- **全知视角**：上帝视角、全景把控、可能产生距离感
- **有限第三人称**：既有距离又能深入一人内心，现代小说最常用
- **视角选择影响**：
  - 读者认同的对象
  - 信息揭示的节奏（悬念、反转）
  - 主题的表达（如多视角暗示真相的多元性）

## 4. 在哪些作品/体裁里出现

### 第一人称经典
- 《了不起的盖茨比》（尼克·卡拉威：旁观者叙述者）
- 《麦田里的守望者》（霍尔顿：主人公叙述者，青少年声音）
- 《洛丽塔》（亨伯特：不可靠叙述者）

### 全知视角经典
- 《战争与和平》（托尔斯泰式全知：哲学介入）
- 《红楼梦》（中国传统全知叙事）
- 《百年孤独》（神话式全知）

### 有限第三人称经典
- 《到灯塔去》（多个人物的内聚焦交替）
- 《尤利西斯》（不同章节不同聚焦）
- 亨利·詹姆斯的大部分作品

### 多视角/复合视角
- 《喧哗与骚动》（四个不同叙述者）
- 《罗生门》/《竹林中》（不可靠的多重叙述）

## 5. 分析方法
- **如何识别**：
  1. 找人称标记（我/你/他她）
  2. 判断叙述者是否在故事中
  3. 判断叙述者知道多少（能否进入他人内心）
  4. 注意视角是否转换

- **如何分析效果**：
  1. 这个视角创造了什么样的距离感？
  2. 读者被引导认同谁？
  3. 什么信息被隐藏了？为什么？
  4. 视角选择如何服务于主题？
