# 例子库（最小工作例子）

## 归一化：`p_\theta(z|x) ∝ const·p(z) => p_\theta(z|x)=p(z)`

**题目**：已知 `p_\theta(z|x) ∝ const·p(z)`，且比例系数对 `z` 是常数（只允许依赖 `x`），说明为什么必有 `p_\theta(z|x)=p(z)`。

**解**：
1) “正比”的严格含义：存在 `C(x)>0`（与 `z` 无关），使得 `p_\theta(z|x)=C(x)p(z)`。
2) 用“概率分布必须归一化”确定 `C(x)`：
   - 连续：`1=∫ p_\theta(z|x)dz = ∫ C(x)p(z)dz = C(x)∫ p(z)dz = C(x)`，所以 `C(x)=1`。
   - 离散：`1=∑_z p_\theta(z|x) = ∑_z C(x)p(z) = C(x)∑_z p(z) = C(x)`，所以 `C(x)=1`。
3) 结论：`p_\theta(z|x)=C(x)p(z)=p(z)`。

> 直觉：如果两个分布在 `z` 上只差一个“与 `z` 无关的常数因子”，那么归一化之后它们完全相同。
