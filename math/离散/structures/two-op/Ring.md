# 环（Ring）

## 1. 一句话
- 同时有加法与乘法的结构：加法像向量加法，乘法像组合；分配律把两者联动。

## 2. 接口：数据 + 公理
- 底层集合：`R`
- 运算：`+ : R×R -> R`，`· : R×R -> R`
- 公理（常用版本）：
  - `(R, +)` 是阿贝尔群（有 `0`、有加法逆元）
  - 乘法结合律：`a·(b·c)=(a·b)·c`
  - 左右分配律：`a·(b+c)=a·b+a·c`，`(a+b)·c=a·c+b·c`
  - 是否要求乘法单位元 `1`：不同教材约定不同；本笔记默认“带 1”的环会特别注明

## 3. 继承与对比
- 子类：交换环、整环、域
- 对比：环里的“可逆”通常只指乘法单位（单位元/可逆元），不等于“每个元素都可逆”

## 4. 例子与反例
- 例子：整数环 `(Z, +, ×)`
- 例子：多项式环 `R[x]`
- 例子：矩阵环 `M_n(R)`（一般不交换）
- 反例：`N` 在 `+`、`×` 下不是环（加法无逆元）

## 5. 常见构造
- 子环：对 `+,-,×` 封闭并包含 `1`（若要求带 1）
- 理想：环里“像子群但能吸收乘法”的子集（见 `math/离散/modules/Subobject.md`）
- 商环：用理想做商（见 `math/离散/modules/Quotient.md`）

## 6. 态射视角
- 环同态：同时保持 `+` 与 `×`（通常还要求保 `1`）
- 核：`ker(f)={r | f(r)=0}`（一定是理想）
