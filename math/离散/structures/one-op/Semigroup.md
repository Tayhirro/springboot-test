# 半群（Semigroup）

## 1. 一句话
- “只有结合律”的乘法结构；是很多代数结构的最小起点。

## 2. 接口：数据 + 公理
- 底层集合：`S`
- 运算：二元运算 `· : S×S -> S`
- 公理：结合律 `a·(b·c) = (a·b)·c`

## 3. 继承与对比
- 子类：`Monoid`（再加单位元），`Group`（再加逆元）
- 对比：半群不要求单位元与逆元

## 4. 例子与反例
- 例子：自然数 `N` 在加法 `+` 下是半群（甚至是幺半群）
- 例子：所有 `n×n` 矩阵在乘法下是半群（一般不可逆）
- 反例：在某集合上随便定义一个运算，若不满足结合律就不是半群

## 5. 常见构造
- 直积：`S×T` 按分量运算是半群

## 6. 态射视角
- 半群同态：`f(a·b)=f(a)·f(b)`

## 7. 速查
- 结合律是“可把括号忽略”的唯一依据
