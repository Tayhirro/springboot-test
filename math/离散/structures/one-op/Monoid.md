# 幺半群（Monoid）

## 1. 一句话
- 半群 + 单位元；“可组合并有不做事的元素”。

## 2. 接口：数据 + 公理
- 底层集合：`M`
- 运算：`· : M×M -> M`
- 公理：
  - 结合律
  - 存在单位元 `e`：`e·a=a·e=a`

## 3. 继承与对比
- 父类：`Semigroup`
- 子类：`Group`
- 容易混淆：单位元是“全局的”，不是每个元素一个

## 4. 例子与反例
- 例子：字符串集合在拼接下是幺半群，单位元是空串
- 例子：`(N, +)` 是幺半群，单位元是 `0`
- 反例：`(Z\\{0}, ×)` 不是幺半群（没有单位元 1？其实 1 在里面，但若换成 `2Z` 在乘法下就没有单位元）

## 5. 常见构造
- 直积：分量运算，单位元是 `(e_M, e_N)`

## 6. 态射视角
- 幺半群同态通常还要求 `f(e)=e`（否则容易丢掉单位结构）
