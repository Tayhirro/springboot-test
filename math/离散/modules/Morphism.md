# 态射：同态 / 同构 / 核 / 像（Morphism）

## 1. 一句话
- “保持结构的映射”是把定义、子对象、商对象、同构定理串成一条线的核心工具。

## 2. 同态（homomorphism）
同态是“保持运算”的映射；不同结构保持的运算不同：
- 群同态：`f(ab)=f(a)f(b)`
- 环同态：`f(a+b)=f(a)+f(b)` 且 `f(ab)=f(a)f(b)`（常见还要求 `f(1)=1`）

## 3. 同构（isomorphism）
- **同构**：可逆同态，且逆映射也是同态。
- 记号：`A ≅ B` 表示“在该结构范畴下同构”（例如群同构/环同构），不是仅仅集合双射。

## 4. 核与像（kernel / image）
它们描述“同态丢掉了什么、留下了什么”：
- 群：`ker(f)={g | f(g)=e}`，`im(f)=f(G)`
- 环：`ker(f)={r | f(r)=0}`，`im(f)=f(R)`

关键性质（会直接喂给同构定理）：
- `ker(f)` 在群里是正规子群；在环里是理想
- `im(f)` 在群里是子群；在环里是子环

## 5. 泛性质口径（为什么“最粗/最细”会出现）
很多构造（直积、商）都可以用一句话概括：
- 直积是“让投影同态成立的最自然对象”
- 商对象是“把核压成 0/单位后得到的最自然对象”

（具体到商群/商环见 `math/离散/modules/Quotient.md`）
