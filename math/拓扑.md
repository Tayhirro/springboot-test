# 拓扑（Topology）学习笔记

> 目标：用“开集/连续/同胚”这套语言，把“形状在连续变形下不变的性质”系统化；并为流形、几何、分析、代数拓扑打底。

## 0. 先修与约定（映射与连续的底层语言）
- 记号：
  - `X, Y`：集合（空间的点集）
  - `P(X)`：`X` 的幂集（所有子集的集合）
  - `f: X -> Y`：映射（函数）
  - `U ⊆ X`：子集；`∅`：空集

### 0.1 映射的基础（后面所有定义都会用到）
- **定义域/陪域**：`f: X -> Y` 表示输入来自 `X`，输出落在 `Y`。
- **像（image）**：`f(A) = {f(x) | x∈A}`（把集合 `A` 里的点推到 `Y`）。
- **原像（preimage / inverse image）**：`f^{-1}(B) = {x∈X | f(x)∈B}`（把 `Y` 里的集合 `B` 拉回 `X`）。  
  注意：这里的 `f^{-1}` 是“原像算子”，不要求 `f` 可逆。
- 原像的基本运算律（非常常用）：
  - `f^{-1}(⋃ B_i) = ⋃ f^{-1}(B_i)`（保任意并）
  - `f^{-1}(⋂ B_i) = ⋂ f^{-1}(B_i)`（保任意交）
  - `f^{-1}(Y\\B) = X\\f^{-1}(B)`（保补集）
- **复合**：`(g∘f)(x) = g(f(x))`。

### 0.2 “连续”是谁的性质？
- **连续是映射的性质**：同一个 `f: X -> Y`，在不同拓扑 `τ_X, τ_Y` 下可能连续也可能不连续。
- 直觉：拓扑决定“哪些集合算开”，而连续的定义就是“开集的原像仍是开集”。

## 1. 为什么需要“拓扑”
- 从几何到拓扑：允许拉伸/弯曲，但不允许撕裂/粘合。
- 研究对象：哪些性质在“连续变形”下保持不变？
- 核心三件事：
  - **开集**：定义“邻域/局部”的语言
  - **连续**：与开集相容的映射
  - **同胚**：拓扑意义下“完全一样”

## 2. 拓扑空间：定义与第一批例子
### 2.1 拓扑的定义
- **拓扑（topology）**：`X` 上一族子集 `τ`（开集族），也就是 `τ ⊆ P(X)`，满足  
  1) `∅, X ∈ τ`  
  2) 任意并仍在 `τ`  
  3) 有限交仍在 `τ`
- **拓扑空间（topological space）**：`(X, τ)`。
- 术语：
  - **开集**：`U ∈ τ`
  - **闭集**：补集是开集
  - **邻域**：包含某个开集的集合（用于“附近”的表达）

### 2.2 基与子基（减少“直接列出所有开集”的负担）
- **基（basis）**：一族集合 `B`，满足覆盖与“交可再细分”，从而生成拓扑。
- **子基（subbasis）**：用有限交生成基，再取任意并生成拓扑。

### 2.3 典型拓扑
- `R` / `R^n` 的标准拓扑（由开球生成）。
- 离散拓扑 / 平凡拓扑：极端例子用来检验定义。
- 子空间拓扑（子集继承“开集”）。

## 3. 连续与同胚：拓扑的“可计算接口”
### 3.1 连续
- **连续（continuous）**：`f: (X, τ_X) -> (Y, τ_Y)` 连续 iff 对任意开集 `V ⊆ Y`（即 `V ∈ τ_Y`），都有 `f^{-1}(V) ∈ τ_X`。
- 常用等价表述（可在此补充证明/练习）：
  - 用闭集的原像
  - 用邻域

- 拓扑的“粗细”会影响连续性（帮助你判断直觉）：
  - 把 `τ_X` 变细（开集变多）通常让 `f` 更容易连续（因为“要求原像是开集”更容易满足）。
  - 把 `τ_Y` 变细（开集变多）通常让 `f` 更难连续（要检查的开集更多了）。

### 3.2 同胚
- **同胚（homeomorphism）**：连续双射且逆映射也连续。
- **同胚不变量**（后续将系统化）：连通性、紧致性、基本群等。

## 4. 由旧空间造新空间：常见构造
- **积拓扑（product topology）**：在 `X×Y` 上取“最粗（开集最少）且使投影连续”的拓扑。  
  记投影 `p_X: X×Y -> X`、`p_Y: X×Y -> Y`，积拓扑要求 `p_X, p_Y` 连续，并且在所有满足这一点的拓扑中它最粗。
- **商拓扑（quotient topology）**：把点按等价关系“粘合”后的最自然拓扑（使投影 `π` 连续）。  
  给定等价关系 `~`，商集 `X/~` 的元素是等价类；投影 `π: X -> X/~` 把点送到它所属的等价类。定义：  
  `U ⊆ X/~` 是开集 ⇔ `π^{-1}(U)` 在 `X` 里是开集。  
  这样 `π` 自动连续；并且这是“最细（开集最多）且使 `π` 连续”的拓扑：任何能让 `π` 连续的拓扑，其开集都必须满足“原像开”，因此不可能比商拓扑更细。
- **粘合/拼接（gluing）**：用商拓扑描述“沿边界贴合”。
- 练习方向：圆 `S^1`、环面 `T^2`、射影空间 `RP^n` 的商表示。

## 5. 三大性质：分离、公理化的“好空间”、以及形状特征
### 5.1 分离公理（Separation axioms）
- `T0, T1, T2(Hausdorff)`：点能否被开集分开？
- 直觉：避免“极限不唯一/点粘不清”的病态。

### 5.2 紧致性（Compactness）
- **紧致（compact）**：任意开覆盖有有限子覆盖。
- 与分析的连接：连续函数在紧致集上取最大最小值（在 Hausdorff 等条件下更好用）。

### 5.3 连通性（Connectedness）
- **连通（connected）**：不能分成两个不交的非空开集。
- **道路连通（path-connected）**：任意两点可由连续曲线连接。
- 区分与关系：`path-connected => connected`（一般不反过来）。

## 6. 度量空间：把“距离”作为一种产生拓扑的方式
- **度量（metric）**：距离函数 `d` 满足非负、对称、三角不等式。
- 度量诱导拓扑：开球生成的拓扑。
- 重要结果：度量空间天然是 Hausdorff、第一可数；与序列极限更贴近。

## 7. 流形的拓扑条件（连接到你的后续学习）
- **拓扑流形**的三条件：Hausdorff、第二可数、局部同胚于 `R^n`。
- 为什么需要“第二可数”：保证空间可管理、可用可数覆盖与拼接工具。
