# 概念图（线性代数）

导航：[math/README.md](../README.md) ｜[math/索引.md](../索引.md) ｜本分支：[math/线性代数/README.md](README.md) ｜[math/线性代数/索引.md](索引.md)

> 读法：先定位对象（向量/映射/矩阵），再看加了什么结构（内积/正交/PSD），最后落到常用分解（谱/SVD/QR）。

## 1. 线性代数主线：向量空间 → 线性映射 → 矩阵
- 向量空间 `V`
  - + 子空间/生成/基/维数 => 坐标表示（选基就是选坐标系）
  - + 线性映射 `T: V -> W` => 核/像/秩-零化度（线性/仿射/多输入“对谁线性”见 [math/线性代数/modules/Linearity.md](modules/Linearity.md)）
  - + 选定基 => 矩阵表示 `A`（同一个映射，不同基下矩阵相似）

## 2. 线性方程组：可解性与最小二乘
- `Ax=b`
  - 高斯消元 => 阶梯形/秩
  - `rank` 决定：解是否存在、自由度多少
- 过定/无解：最小二乘 `min_x ||Ax-b||^2`
  - 需要内积/范数（通常在 `R^n`/`C^n`）

## 3. 内积/正交：让“几何”可计算
- 向量空间 + 内积 `<·,·>` => 正交/投影/角度/长度
- Gram–Schmidt => 正交归一基
- 正交投影 + 最小二乘
- QR 分解：把消元/投影结构化

## 4. 谱与分解：把矩阵变“可读”
- 特征值/特征向量：`Av=λv`
- 对称/自伴随矩阵 => 谱定理（可正交对角化）
- 一般矩阵 => SVD（最通用、最稳健）

## 5. PSD 与 Gram：核方法的底座
- 半正定矩阵（PSD）`A \succeq 0` <=> `x^T A x ≥ 0`（对所有 `x`）。见 [math/线性代数/modules/PSD.md](modules/PSD.md)
- Gram 矩阵：`K_{ij} = <v_i, v_j>` 一定 PSD
- 核函数（kernel）：`k(x,y)` 让 `K_{ij}=k(x_i,x_j)` 总是 PSD  
  见 [math/线性代数/modules/KernelMethods.md](modules/KernelMethods.md)
