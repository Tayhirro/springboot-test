# 概念图（泛函分析）

## 0. 读法（先看“底座”，再看“加了什么结构”）
- 很多“空间”并不是互斥分类，而是：同一个集合/线性空间上叠加不同结构。
- 下面用依赖箭头写成：`A + 结构 => B`（在 A 上加结构得到 B），以及 `B => A`（B 自动带来 A）。

## 1. 泛函分析主线：线性 + 范数/内积
- 线性空间（vector space）
  - + 范数 `||·||` => 赋范空间（Normed space）
    - + 完备（每个柯西列都收敛） => Banach 空间
  - + 内积 `<·,·>` => 内积空间（Inner product space）
    - + 完备（按内积诱导范数） => Hilbert 空间

关键蕴含（必记）：
- 内积空间 => 赋范空间（`||x|| = sqrt(<x,x>)`）
- 赋范空间 => 度量空间（`d(x,y)=||x-y||`）
- Hilbert 空间 => Banach 空间

## 2. 拓扑/收敛主线：从“开集”到“距离”
- 集合 `X`
  - + 开集族 `tau` => 拓扑空间（Topological space）
  - + 距离 `d` => 度量空间（Metric space） =>（自动）拓扑空间

子集继承（常用默认约定）：
- `Y ⊂ X`：把 `X` 的拓扑限制到 `Y` 得到子空间拓扑；若 `X` 是度量空间，也可把 `d` 限制到 `Y` 得到 `Y` 的度量与拓扑。

介于度量与拓扑之间的统一结构（常用于“一致连续/完备化”）：
- 度量空间 => 一致空间（Uniform space） => 拓扑空间

线性与拓扑叠加（在泛函分析里很常见）：
- 线性空间 + 拓扑，且 `+`、`scalar mult` 连续 => 拓扑向量空间（TVS）
- 赋范空间 =>（自带）TVS（因为范数诱导拓扑，且运算连续）

## 3. 测度/积分主线：函数空间的底座
- 集合 `Omega`
  - + sigma-代数 `Sigma` => 可测空间（Measurable space）
    - + 测度 `mu` => 测度空间（Measure space）
      - + `mu(Omega)=1` => 概率空间（Probability space）

与线性结构结合后出现的核心函数空间：
- 测度空间 + “几乎处处相等的等价类” + `p`-范数 => `L^p(Omega)`（通常是 Banach；`p=2` 是 Hilbert）

## 4. 光滑/微分主线：把“导数”带进来（PDE 常用）
- 集合 `M`
  - + 拓扑 `tau` => 拓扑空间
    - +（Hausdorff + 第二可数 + 局部同胚于 `R^n`）=> 拓扑流形（topological manifold）
      - + 可微/光滑结构（兼容图册）=> 可微/光滑流形（smooth manifold）
        - + 黎曼/伪黎曼度量（在每点切空间上给内积并随点光滑变化）=>（伪）黎曼流形（能谈长度、测地线、曲率、Laplace-Beltrami 等）

和 Hilbert 空间的关系（一个典型例子）：
- `H = R^3`（标准内积）本身就是一个 3 维 Hilbert 空间（也是度量/拓扑空间）。
- `S^2 = {x∈R^3 : ||x||=1}` 是嵌入在 `H` 里的 2 维光滑流形（曲面）；其拓扑就是从 `R^3` 继承的子空间拓扑（等价于限制度量 `d(x,y)=||x-y||` 得到的拓扑），并且每点附近都“看起来像”一块 `R^2`。

从“微分”到“积分”的一个常见依赖链：
-（伪）黎曼流形 +（定向）=> 体积形式 `dV` => 自然测度 `mu` => 能在流形上积分

与函数空间结合（Sobolev）：
- 开集 `Omega`（`R^n` 的开子集）+ 测度 + 弱导数 + `L^p` => Sobolev `W^{k,p}(Omega)`（`p=2` 常记 `H^k`）
-（扩展）光滑流形 `M` + 测度（如 `dV`）+ 弱导数（用坐标/联络定义）=> Sobolev `W^{k,p}(M)`

## 5. 其他常见“结构轴”（与线性/拓扑可叠加）
- 顺序（order）：集合 + 偏序 `<=` => 偏序集（可进一步诱导序拓扑）
- 代数（algebra）：集合 + 运算（群/环/域/代数）
  - + 拓扑 => 拓扑群/拓扑环
  - + 光滑结构 => 李群（Lie group）

## 6. 算子与对偶（把“几何”变成“可算对象”）
- 线性泛函：`f: X -> R/C`（在对偶空间 `X*` 里）
- 线性算子：`T: X -> Y`
- 关键结构：
  - 对偶 `X*`、算子范数、伴随 `T*`
  - 弱/弱* 拓扑（用对偶测试）

## 7. 工具定理（常用顺序）
- Hahn–Banach：延拓/分离（最先用）
- Banach–Steinhaus（一致有界）：从逐点有界推出一致有界
- 开映射 / 闭图像：把“连续双射/图像闭”变成结构结论

## 8. 与后续主题的连接
- 谱理论（算子谱、紧算子）
- PDE（Sobolev、弱解、变分法）
