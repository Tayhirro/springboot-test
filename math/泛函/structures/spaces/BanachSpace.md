# Banach 空间（Banach Space）

## 1. 一句话
- “完备”的赋范空间：所有柯西列都收敛，这让极限操作不会跑出空间。

## 2. 定义
赋范空间 `X` 称为 Banach，如果在度量 `d(x,y)=||x-y||` 下是完备的。

## 3. 例子与反例
- 例子：`R^n`、`C([0,1])`（一致范数）、`L^p`（`1 ≤ p ≤ ∞`）
- 反例：多项式在 `C([0,1])` 的一致范数下不是闭的（其完备化是 `C([0,1])`）

## 4. 速查
- 三大定理：开映射/闭图像/一致有界（见 `math/泛函/modules/Theorems.md`）
