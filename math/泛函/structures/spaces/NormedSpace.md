# 赋范空间（Normed Space）

## 1. 一句话
- 在线性空间上加“长度/大小”的概念，从而能谈连续、极限、收敛与算子范数。
-距离 的概念


## 2. 定义（数据 + 条件）
给定线性空间 `X`（实/复），范数 `||·||: X -> R` 满足：
- 非负与分离：`||x|| ≥ 0` 且 `||x||=0 <=> x=0`    可区分不同点的   距离  
- 齐次：`||αx|| = |α|·||x||`                     误差不变
- 三角不等式：`||x+y|| ≤ ||x|| + ||y||`           距离长度更长

由范数诱导度量：`d(x,y)=||x-y||`。

## 3. 例子
- `R^n` 配欧几里得范数/`l^1`/`l^∞` 范数
- 连续函数空间 `C([0,1])` 配一致范数 `||f||_∞`
