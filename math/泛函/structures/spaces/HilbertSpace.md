# Hilbert 空间（Hilbert Space）

## 1. 一句话
- 完备内积空间；“正交分解 + 极限”能稳定共存，是投影、最小二乘、谱理论的标准舞台。

## 2. 定义（内积 + 完备）
给定内积空间 `H`，由内积诱导范数 `||x|| = sqrt(⟨x,x⟩)`，再诱导度量 `d(x,y)=||x-y||`。
如果 `H` 在这个度量下是**完备（complete）**的，则称 `H` 为 **Hilbert 空间**。

### 2.1 完备
- **柯西列（Cauchy sequence）**：序列 `x_n` 满足：对任意 `ε>0`，存在 `N`，使得当 `m,n≥N` 时 `||x_m - x_n|| < ε`。
  - 直觉：从某一项开始，序列后面的点互相都非常接近，像是在“逼近某个极限”。
- **完备**：每个柯西列都在 `H` 中收敛到某个 `x∈H`。

因此，Hilbert 空间等价地说是：
- “内积空间里，每个柯西列（按内积范数）都有极限且极限仍在空间里”。

## 3. 例子
- `R^n`、`C^n`
- `l^2`（平方可和序列）ℓ2={(x1​,x2​,x3,…) : xn∈C（或 R）, n=1∑∞∣xn∣^ 2<∞}
- `L^2(Ω)`（平方可积函数等价类）

### 3.1 例子背后的关键点
- `R^n`、`C^n`：有限维内积空间都自动完备（所以一定是 Hilbert）。
- `l^2`：内积 `<x,y> = sum_{n=1}^∞ x_n * conj(y_n)`，在 `||·||_2` 下完备。
- `L^2(Ω)`：内积 `<f,g> = ∫_Ω f(x) * conj(g(x)) dμ(x)`，在 `||·||_2` 下完备。

## 4. 典型能力（为什么“完备”在这里是关键）
- **投影定理（Projection theorem）**：对任意闭子空间 `W ⊆ H`，每个 `x∈H` 都能唯一分解为 `x = w + w_perp`，其中 `w∈W` 且 `w_perp ⟂ W`；并且 `w` 是到 `W` 的最近点。


x - w' = (x - w) + (w - w') = e + (w - w')
||x - w'||^2 = ||e + (w - w')||^2 = ||e||^2 + ||w - w'||^2 ≥ ||e||^2


- **正交基/展开**：在很多 Hilbert 空间里可以用正交归一基做“傅里叶式展开”，把问题转成系数问题。

（这些结论在一般内积空间里可能失败，失败点往往就是“不完备导致最近点/极限不在空间里”。）

## 5. 速查
- 关系：Hilbert ⇒ Banach（完备赋范空间），但反过来不成立（Banach 不一定来自内积）。
- Riesz 表示（对偶）：`H* ≅ H`（见 `math/泛函/modules/Duality.md`）
