# 对偶与表示（Duality）

## 1. 一句话
- 对偶空间把“点的几何”转成“线性测试函数”的语言，是弱拓扑与表示定理的入口。

## 2. 对偶空间
对赋范空间 `X`：
- `X*`：所有连续线性泛函 `f: X -> R/C` 的集合
- 范数：`||f|| = sup_{||x||≤1} |f(x)|`

## 3. Hilbert 空间的 Riesz 表示（占位）
在 Hilbert 空间 `H` 中，每个连续线性泛函都可写成内积形式（`H* ≅ H`）。

## 4. 速查
- 弱拓扑/弱收敛用 `X*` 来定义（见 `math/泛函/modules/Convergence.md`）
